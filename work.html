A few things that I have worked on...

I have worked on a variety of projects at various employers, using a wide range of skills. Scroll down for a few examples of my work.

Citizen website and portal

This project was to build the citizen website and portal for Barnet Council, closely following GDS guidelines for usability and accessibility.

I had various roles across this project, largely as the lead front-end developer, but also designer and UX. Main main role was to build HTML templates, create the corresponding styles and maintain the style guide. I also made many user centered design decisions, firstly alongside a UX designer, then as the sole UX resource. This was an agile run project where I worked alongside UX designers/researchers, developers, testers, project managers, and analysts to help build barnet.gov.uk piece by piece in a continuous release cycle.

www.barnet.gov.uk

Car insurance quote pages

In this project, I built the quote page for the Tesco car insurance site, integrating complex functionality and code.

I worked in a purely front-end developer capacity for this project, working on a Java based platform to build the templates and client side functionality. This included complex JavaScript code, HTML and LessCSS based template building, and incorporating the functionality with the existing back-end setup. This was a self managed project in a waterfall style, where I had a set of requirements and a deadline by which to finish.

www.tescobank.com/car-insurance/

Salary guide

For this project, we were asked to restyle the Reed Salary Guide, so I was asked to provide concepts for a "refresh" of the current design.

My role in this was to provide various concepts (created in Photoshop) to refresh the look of the existing salary guide, whilst not changing the core functionality or creating too much change for the developers. I put forward a new color scheme and some redesigned assets which the client was happy with, and the design was integrated shortly after.

www.reedglobal.com

Full website

A personal project for a client, to create a small website to showcase their takeaway and menu

I was full stack designer/developer whilst working on this, from designing concepts for sign off from the client to full build and release. This site was design fully responsive after checking analytics on the previous site and discussing with the client that the majority of the sites users were using a handheld device. This site now attracts over 500 users a month and over a 1,000 page views, which we have considered a huge success for this small business.

www.alessios.co.uk

Motorbike insurance

Maintenance of a high-use insurance site for a high value client.

I was solely responsible for the maintenance of this and several other insurance websites, implementing any changes requested by the client. This encompassed all contact with the client, estimating, development, and liaising with testers and project managers to achieve sign off. I personally maintained and managed these sites very successfully for over 10 months, swiftly turning around live bugs and quickly implementing requesting amends.

www.rac.co.uk

Site login

Lead Design and maintenance of large platform webapp.

For several months I was the lead designer/front-end developer on this site and various micro sites associated with it. I mainly implemented code changes to templates and style changes, but also oversaw a full re-brand and the implementation of a brand new login functionality within the site. Alongside these responsibilities I also maintained the site, clearing up all raised bugs and ensuring there were no style or client-side issues.

www.connect.innovate.uk
